Lesson2: Declare, Usage Variable & Constant¶

  • Biến (Variable) là vùng nhớ dùng để lưu trữ các giá trị của chương trình và có thể thay đổi giá trị.
  • Constant (hằng số) là một giá trị không thể bị thay đổi bởi chương trình trong quá trình thực thi thông thường (a constant is a value that cannot be altered by the program during normal execution)
Declare ES5 ES6
Variable var
e.g var myVar='Hello';		 let
e.g let myVar='Hello';
Constant const
e.g const PHONENUM ='0903642155';		 const
e.g const MY-TEXT ='unchangeable';

Practice¶

  1. create a file with name VariableConstant.js
    // Declare Variable

//Declare Constant

console.log(PI);

NOTE TO DECLARE VARIABLE NAME AND CONSTANT NAME
1. Variable name & Constant name must be meaning
2. Variable name is started with the lower case
e.g. myName, myCount -> follow-up Camel Care formal
3. Constant name is declarated with upper case
e.g. MYNAME, PI, MY_NAME
4. The variable name, the constant constant are not duplicated with key-word of a program

Lesson3: String¶

What's String?

A string can be any text inside double or single quotes String indexes are zero-based: The first character is in position 0, the second in 1, and so on.

Sample:
let myName = "Volvo XC60";
Let myname = 'Volvo XC60';

String Properties

Property Description
length Returns the length of a string included of blank

String Methods

Methods Description
split Splits a string into an array of substrings
concat Combine two or more strings, and returns a new combined strings
indexOf Returns the position of the first found occurrence of a specified value in a string
