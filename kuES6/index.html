  • Execute test for kintoneUtiliti by ES6 (2018/08/01)

  1. Get source at https://github.com/kintone/kintoneUtility

  2. Create new folder e.g testkubyES6

  3. Create a new folder "dist" into "testkubyES6" for containing archives

  4. Create a new file into "testkubyES6" with content the e.g test.js

    import kintoneUtility '../kintoneUtility';
console.log(kintoneUtility);

  5. Copy & paste files into "testkubyES6"

    • webpack.config.js
    • package.json

  6. Edit the content "webpack.config.js" file

    Before
    entry: {
        'kintoneUtility': './src/js/main.js',
        'kintoneUtility.min': './src/js/main.js'
    },
⇩
After
    entry: {
        'kintoneUtility': './src/js/test.js',
        'kintoneUtility.min': './src/js/test.js'
    },

  7. Edit the content "package.json" file

    Remove the 2 lines
"eslint": "^4.6.1",
"eslint-config-kintone": "^1.2.0",

  8. Run the comment in edit

    npm install
   ==> confirm the libs are installed in the new folder name "" which is created automaticaly 
npm run build
   ==> there are 2 files that are output at
   ./dist/kintone-ui-component.min.js
   ./dist/kintone-ui-component.min.css

  9. Create an application on kintone then add 2 files that are created by runnnung "npm run build" on "JavaScript and CSS Customization"

  10. Access into application and view console log Result: "kintoneUtility" object is output => successful.
